# CyberLens - 全能网络空间测绘工具

CyberLens 是一款集成了 **FOFA**、**Quake (360)**、**Hunter (鹰图)** 和 **ZoomEye (钟馗之眼)** 四大测绘引擎的综合网络安全工具。它不仅支持多源数据的聚合搜索，还内置了强大的本地扫描与资产管理功能，是红队评估、资产收集和漏洞挖掘的得力助手。

---

## 🚀 核心功能

### 1. 🌐 多源测绘聚合
*   **四大引擎支持**: 同时接入 FOFA、Quake、Hunter、ZoomEye API。
*   **智能语法转换**: 使用一套通用语法（如 `app="Apache"`）即可查询所有引擎，无需记忆各自复杂的特定语法。
*   **数据聚合去重**: 自动合并多源结果，去除重复资产，提供最全的资产列表。
*   **语法自动补全**: 搜索框内置语法提示，降低使用门槛。

### 2. 🔍 深度资产扫描
*   **端口扫描**:
    *   **Nmap 级识别**: 内置完整的 Nmap 指纹库，精准识别服务与版本。
    *   **灵活配置**: 支持 Top 100、Web常用、数据库、企业内网或自定义端口范围扫描。
*   **Web 指纹识别**:
    *   **海量指纹**: 内置数万条 CMS、框架、设备指纹规则。
    *   **自动联动**: 端口扫描发现 Web 服务后自动触发指纹识别。
    *   **可扩展**: 支持加载外部 JSON 指纹库 (`data/finger.json`)。
*   **JS 爬虫**:
    *   **深度爬取**: 自动爬取页面链接，并深入分析 JS 文件提取隐藏接口。
    *   **信息提取**: 自动提取页面标题、状态码、响应长度。
*   **目录扫描**:
    *   **敏感路径**: 内置字典扫描常见敏感目录（如 `.git`, `/admin` 等）。
    *   **自定义字典**: 支持加载外部字典文件 (`data/dict.txt`)。

### 3. 📂 本地资产管理
*   **本地导入**: 支持粘贴 URL 列表导入本地目标。
*   **混合处理**: 导入的本地资产可与搜索结果一样，无缝使用所有扫描功能。
*   **结果清洗**:
    *   **移除 404**: 一键清理无效目标。
    *   **标题去重**: 快速剔除重复站点。
*   **数据导出**: 支持导出 Excel/CSV 格式报告。

## � 使用说明

### 1. 环境准备
*   需安装 **Java 8** 或更高版本运行环境。
*   首次运行会自动在根目录生成 `data` 文件夹。

### 2. 配置 API
1.  启动程序，点击右上角 **“设置”**。
2.  输入您拥有的测绘引擎 API Key（FOFA、Quake、Hunter、ZoomEye）。
3.  (可选) 配置 HTTP/SOCKS 代理以加速访问。

### 3. 开始搜索
*   在搜索框输入查询语句，例如：
    *   `domain="example.com"` (搜索域名)
    *   `ip="1.1.1.1/24"` (搜索C段)
    *   `title="后台管理"` (搜索标题)
*   点击 **“搜索”** 按钮或回车，结果将实时显示在下方表格中。

### 4. 资产操作
*   **右键菜单**: 在结果表格中右键点击任意条目，可进行：
    *   **打开链接**: 浏览器访问目标。
    *   **端口扫描**: 对当前 IP 进行端口探测。
    *   **C段查询**: 快速反查该 IP 所在 C 段资产。
    *   **爬虫/目录扫描**: 对当前 URL 进行深度探测。
*   **批量操作**: 使用顶部工具栏的 **“本地导入”** 或 **“批量搜索”** 功能。
